# Cleanup Scripts

Скрипты для очистки временных сайтов с истекшим TTL.

## Назначение

Эти скрипты выполняют:
- Удаление временных сайтов с истекшим сроком действия
- Очистку связанных файлов (логотипы, превью)
- Логирование удаленных записей

## Использование

Скрипты могут быть запущены:
- Вручную
- Через cron
- Как scheduled tasks
